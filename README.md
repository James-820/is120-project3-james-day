# Project 3 - James Day

For my project, I make a lift tracker that allows you to keep track of which lift was done at which weight, and for how many repetitions and sets. I learned how to use the `.split()` method for JavaScript, which helped me store the reps in a JSON format in Supabase. I think the most difficult parts were first, figuring out a good way to store the reps in the Supabase table, and second, figuring out how to get the HTML elements from the table entries. However, the second one was easier, in my opinion. Figuring out a way to both get reps/sets from the user and store them was tricky.

In the future, I would definitely change the styling and put a lot more effort into it. I could also look up how to configure logins with Supabase and allow each user to keep track of their own lifts. I imagine that would also involve a TON of code refactoring to make it work.
